Introduction to Computers

Course Overview

The course is meant to give a basic understanding of what computers are as well as basic software development skills. It provides an opportunity to have a practical experience in development of basic interactive websites. Students are acquainted with tools like git, VMs, docker, VS Code

Syllabus

Introduction to Computers, Bits, Bytes, Binary, Computer Hardware, Operating Systems, Linux commands, Computer Networks, Internet, Protocols, version control, HTML, Creating a Webpage, CSS, Advanced Concepts in Webpage Creation, Webpage Layout, Javascript, HCI and Website Design, Computer Security, Privacy and Big Data, AI, ML, Cloud Computing.

Learning Outcome

  • Understand the basic building blocks of Computer Software, Hardware, and Networks.
  • Create a Website using HTML, CSS, Javascript
  • Learn modern concepts in Computer Science.
